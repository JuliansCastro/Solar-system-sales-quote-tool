# Solar-system-sales-quote-tool
☀️ Herramienta web para diseño y cotización de sistemas de energía solar fotovoltaica. Dimensionamiento On-Grid/Off-Grid con integración PVGIS, gestión de clientes/proyectos, inventario de equipos, cotizaciones con PDF/Excel, proyecciones financieras a 25 años y dashboard interactivo. Django + Tailwind CSS + Plotly.js
